加密网站 解密网站

Public Key Cryptography Demonstration

两个网站演示 RSA：发送方加密，接收方解密

本项目适合长期部署到 GitHub Pages。加密网站只使用公钥，解密网站使用私钥， 更贴近真实的信息传递流程。

进入加密网站 进入解密网站
01

加密网站

输入明文后，浏览器调用加密库和 RSA 公钥生成 Base64 密文。

src/rsa-encrypt-lib.js
02

密文传递

密文可以复制，也可以通过页面按钮带到解密网站进行现场演示。

localStorage / clipboard
03

解密网站

粘贴密文后，浏览器调用解密库和 RSA 私钥还原明文。

src/rsa-decrypt-lib.js
算法：RSA-OAEP / SHA-256 / 2048-bit 说明：内置私钥仅用于课堂演示，不用于真实业务。
